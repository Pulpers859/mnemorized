VISUAL METAPHOR LIBRARY
=======================
Central catalog of proven visual metaphors for Mnemorized prompt generation.
Referenced by gemini-constitution.txt (Visual Encoding Hierarchy).
Metaphors must be consistent across all topics — if banana = potassium here,
it means potassium everywhere. Add new entries after stress-test validation.

Last updated after DKA mixed-device + metaphor stress test.

═══════════════════════════════════════════════════════════════════════════
USAGE RULES
═══════════════════════════════════════════════════════════════════════════

1. One metaphor per concept, globally. Never reuse a metaphor for a
   different meaning in another topic.
2. Metaphor objects should be OVERSIZED and visually prominent — Sketchy
   style. A banana barrel, not a single small banana.
3. When using a pure metaphor (no text label), the object must be
   unambiguous in context. If a viewer might confuse it, add a small
   hybrid label.
4. New metaphors should be stress-tested before adding to this library.
5. Cultural universality matters — prefer metaphors that work across
   backgrounds (banana = potassium is global medical culture).
